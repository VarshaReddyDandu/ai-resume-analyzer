# AI Resume Bullet Generator

An AI-powered web application that generates high-impact resume bullet points using LLaMA 3 via Groq.

## 🚀 Features
- Real-time AI-powered bullet generation
- FastAPI backend
- Streamlit frontend
- Free LLM integration (Groq)
- Download generated bullets

## 🧠 Tech Stack
- Python
- FastAPI
- Streamlit
- Groq (LLaMA 3)

## ▶️ Run Backend
uvicorn app:app --reload

## ▶️ Run Frontend
streamlit run ui.py
